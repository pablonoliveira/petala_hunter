Atalhos desta versão:

1. Logs de debug no terminal
- Ativos por padrão com DEBUG_LOGS=True.
- Para desativar no Windows PowerShell:
  $env:DEBUG_LOGS="False"

2. Fallback para Meta Ads Library
- Primeira tentativa: best[ext=mp4]/best
- Segunda tentativa: bestvideo+bestaudio/best

3. Dependências
- pip install -r requirements.txt

4. Execução
- python app.py
